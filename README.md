# zustand_todo_list

## Скрипты

```
# Install
npm i                    # install dependencies

# General
npm run dev            # run app in the development mode
```

Open `http://localhost:3000`

## Технологии

- React js
- Zustand
- TypeScript
- ViteJS
- LocalStorage

## Функционал

- Добавление задачи
- Удаление задачи
- Редактирование задачи
- Завершение задачи
- Список выполненных задач
